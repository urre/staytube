## Install

```
git clone https://github.com/urre/staytube.git
cd staytube
npm install
```

## Run the dev server:

Note:  A ``.config.js`` is needed with a valid YouTube V3 API Key

```
npm start
```

Then open [localhost:8080](http://localhost:8080/).
